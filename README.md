\# Java Programming Masterclass 



A comprehensive collection of Java programs covering core concepts, Object-Oriented Programming (OOPS), and Data Structures \& Algorithms (DSA).



\##  Project Structure



\- \*\*OOPS/\*\*: Deep dive into Abstraction, Encapsulation, Inheritance, and Polymorphism.

\- \*\*30\_question\_to\_solve/\*\*: DSA practice including Kadane’s Algorithm, Array rotations, and more.

\- \*\*binary\_search/\*\*: Implementation of various searching logic.

\- \*\*constructor/\*\*: Detailed examples of constructor chaining and overloading.



\##  Concepts Covered

\* \*\*Core Java:\*\* Loops, Arrays, Recursion, and Static vs Non-Static.

\* \*\*OOPS:\*\* Real-world examples like "The Iron Vault" (Encapsulation) and "Ola App" (Inheritance).

\* \*\*Problem Solving:\*\* 50+ Java files focused on logic building and competitive programming.



\##  How to Run

1\. Clone the repository:

&nbsp;  `git clone https://github.com/srinivasan2113/Java-Programming-Masterclass.git`

2\. Compile any file: `javac FileName.java`

3\. Run: `java FileName`
